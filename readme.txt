In the BrightData Console, when creating a Amazon Scraper please do the following:

1) Add an 'search' input into the input schema for Step 1
2) Add Step 1 Interaction code & Parser code
1) Add an 'url, search, id' input into the input schema for Step 2
3) Add Step 2 Interaction code & Parser code
4) Save to development and test
5) Save to production
